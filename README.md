# D2C Customer Churn Intelligence - Part 1

## Objective
Perform data auditing, exploratory data analysis, and business understanding for customer churn prediction.

## Files
* eda_audit.ipynb
* data_quality_report.md
* business_memo.md

## Dataset
The project uses customer, order, support ticket, web activity, intervention, and churn label datasets.

## How to Run
1. Install dependencies
```bash
pip install -r requirements.txt
```

2. Open and execute:
```bash
jupyter notebook eda_audit.ipynb
```

## Outputs
* Data quality assessment
* Exploratory data analysis
* Churn-risk hypotheses
* Business recommendations
